# コードレビュー実行

汎用コードレビューエージェントを使用して、以下の観点から包括的にレビューします：

## レビュー観点
- コード品質（可読性、保守性、一貫性）
- アーキテクチャと設計（SOLID原則）
- パフォーマンス（効率性、スケーラビリティ）
- セキュリティ（OWASP Top 10）
- テスト（カバレッジ、テストの質）
- ドキュメント（README、コメント、API仕様）

## 指示
`.claude/agents/code-reviewer.md` の定義に従って、指定されたコードをレビューしてください。
レビュー結果は、以下のフォーマットで出力してください：

1. 総合評価（品質スコア、推奨アクション）
2. 良い点のリスト
3. 改善が必要な点（カテゴリ別、ファイルパスと行番号付き）
4. 提案・アドバイス
5. チェックリスト

レビュー対象が指定されていない場合は、最近変更されたファイルまたは現在のPRを対象としてください。
