# KumoCodeを始めよう

KumoCodeへようこそ！このチュートリアルでは、KumoCodeの基本的な使い方を学びます。

## はじめに

KumoCodeは、Markdownで記事を書くだけで美しいドキュメントサイトを作成できるプラットフォームです。

### 必要な知識

- 基本的なMarkdown記法
- テキストエディタの使い方

## フロントマター

各記事の先頭には、**フロントマター**と呼ばれるメタデータを記述します。

```yaml
---
title: 記事のタイトル
date: 2025-11-16
tags: [Tag1, Tag2, Tag3]
---
```

### フロントマターのフィールド

- **title**: 記事のタイトル（必須）
- **date**: 公開日（YYYY-MM-DD形式、必須）
- **tags**: タグの配列（オプション）

## 見出し

見出しは`#`記号で表現します。

```markdown
# h1見出し（最大）
## h2見出し
### h3見出し
#### h4見出し
##### h5見出し
###### h6見出し（最小）
```

## テキストの装飾

### 強調

テキストを強調するには、以下の記法を使用します：

- **太字**: `**太字**` または `__太字__`
- *斜体*: `*斜体*` または `_斜体_`
- ***太字かつ斜体***: `***太字かつ斜体***`

### 取り消し線

~~取り消し線~~は`~~テキスト~~`で表現します。

## リスト

### 順序なしリスト

```markdown
- アイテム1
- アイテム2
  - ネストしたアイテム
  - もう一つのネストしたアイテム
- アイテム3
```

実際の表示：

- アイテム1
- アイテム2
  - ネストしたアイテム
  - もう一つのネストしたアイテム
- アイテム3

### 順序付きリスト

```markdown
1. 最初のステップ
2. 次のステップ
3. 最後のステップ
```

実際の表示：

1. 最初のステップ
2. 次のステップ
3. 最後のステップ

## リンク

リンクは`[リンクテキスト](URL)`の形式で記述します。

例: [KumoCode公式サイト](https://github.com/Sunwood-ai-labs/KumoCode)

## 引用

引用は`>`記号で表現します。

```markdown
> これは引用文です。
> 複数行にわたることもできます。
```

実際の表示：

> これは引用文です。
> 複数行にわたることもできます。

## 水平線

水平線は`---`で表現します。

---

## インラインコード

インラインコードは`` `バッククォート` ``で囲みます。

例: `console.log("Hello, KumoCode!")`
